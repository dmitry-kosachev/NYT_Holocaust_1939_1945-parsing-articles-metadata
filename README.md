# Описание репозитория

Этот репозиторий представляет собой исследование, направленное на сбор метаданных статей газеты The New York Times периода 1939—1945 годов, посвящённых преследованиям еврейского населения нацистским режимом. Метаданные собирались посредством API The New York Times, используя ключевые слова («Jew», «Jews», «Jewish») и соответствующие тематики статей (например, «MINORITIES AND OPPRESSED GROUPS», «OCCUPIED COUNTRIES», «ATROCITIES»). Исследование позволяет проанализировать освещение Холокоста американской прессой в годы Второй мировой войны.

# Repository Description

This repository contains a research project focused on collecting metadata of articles from The New York Times during the period 1939–1945, covering the persecution of Jews by the Nazi regime. The metadata was gathered using The New York Times API with keywords such as “Jew,” “Jews,” and “Jewish” in articles related to specific subjects including “MINORITIES AND OPPRESSED GROUPS”, “OCCUPIED COUNTRIES”, and “ATROCITIES”.
